# Sapphire Skin Medical Wellness Website

This is the official website for Sapphire Skin Medical Wellness.

## Setup & Running
1.  **No installation required.** This website uses standard HTML, CSS, and JavaScript.
2.  Simply double-click `index.html` to open it in your web browser.

## Customization
-   **Logo**: The website uses the logo found at `IMG/LOGO.jpeg`.
-   **Hero Image**: The main background image is set to `IMG/IMG_1603.jpeg`. If you want to change it, replace that file or update the CSS in `index.html` (look for `.hero-bg`).
-   **Treatment Images**: Currently, the website uses placeholder images. To use your own high-quality photos:
    1.  Place your images in the `IMG` folder.
    2.  Open `index.html` in a text editor (like Notepad or VS Code).
    3.  Search for `https://placehold.co` and replace those URLs with your local image paths (e.g., `IMG/facial.jpg`).

## Technologies Used
-   HTML5
-   Tailwind CSS (via CDN)
-   JavaScript (Vanilla)
-   FontAwesome (Icons)
-   Google Fonts (Outfit & Playfair Display)
